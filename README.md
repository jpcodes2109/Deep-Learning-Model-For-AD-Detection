# Deep Learning Model for Alzheimer's Disease Detection

## Overview

This project implements a Deep Learning-based system for detecting Alzheimer's Disease (AD) from brain MRI images. The model analyzes MRI scans and predicts the stage of Alzheimer's Disease using advanced deep learning techniques.

## Features

- MRI image preprocessing
- Deep learning-based classification
- Alzheimer's stage prediction
- Model training and evaluation
- Visualization of training performance
- Prediction on new MRI images

## Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab


3. Install required dependencies.

4. Run all cells sequentially.

5. Upload an MRI image for prediction.

## Dataset

The model is trained on Alzheimer's MRI datasets containing multiple disease stages. Images are preprocessed before training to improve model performance.

## Results

The model can classify brain MRI images and assist in the early detection of Alzheimer's Disease. Deep learning enables automated analysis and can support healthcare professionals in diagnosis.
